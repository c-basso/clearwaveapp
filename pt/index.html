Perguntas Frequentes

Isso vai danificar meus alto-falantes?

Não, as ondas sonoras são seguras e não danificarão seus alto-falantes. É basicamente o mesmo que tocar música no seu telefone, mas com frequências cuidadosamente calibradas projetadas especificamente para ejeção de água.

Isso funciona em todos os dispositivos?

Clear Wave funciona em dispositivos iPhone e iPad. O app é otimizado para iOS e fornece os melhores resultados em hardware Apple. Requer iOS 15.1 ou posterior.

Por quanto tempo devo tocar o som para remover a água?

Geralmente 10-30 segundos é suficiente, mas você pode repetir o processo se necessário. Você pode testar o volume do seu alto-falante após cada ciclo para verificar se o som melhorou.

E se a ejeção de água não funcionar?

Se a ejeção de água não funcionar imediatamente, tente repetir o processo 2-3 vezes. Para água teimosa, você também pode tentar o gerador de tons de frequência em diferentes frequências. Se o problema persistir, pode indicar dano de hardware que requer reparo profissional.

O app é gratuito para usar?

Clear Wave é gratuito para baixar com recursos básicos de ejeção de água. Recursos premium como o gerador de tons avançado estão disponíveis através de compras no app para usuários avançados.

Posso usar isso para outros testes de áudio?

Sim! Clear Wave inclui ferramentas de áudio profissionais para testar audição, afinar equipamentos de áudio, monitorar níveis de ruído e verificar o equilíbrio de canais estéreo - tornando-o um kit de ferramentas de áudio completo.
